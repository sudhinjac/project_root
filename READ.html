📊 Stock Analysis App

Architecture overview and run guide

Overview

Scalable stock analysis application with FastAPI backend and Streamlit frontends. Basic app: annual return, volatility, VaR, RSI, MACD. Comprehensive app: financials (P&L, Balance Sheet, Cash Flow), technicals (CAGR, Sharpe, Monte Carlo, ARIMA, HMM), sentiment, and AI analyst perspective.

Architecture

Layered design with dependency injection and repository pattern:

├── app/
│   ├── main.py                    # FastAPI app entry point
│   ├── domain/

Dependencies

Core: FastAPI, Uvicorn, Streamlit, Plotly, Pandas, NumPy, SciPy, yfinance, requests, httpx.

Comprehensive: feedparser, nltk, statsmodels, scikit-learn, hmmlearn, ollama, beautifulsoup4, matplotlib, vaderSentiment.

How to Run

1. Install dependencies

cd project_root
pip install -r requirements.txt

2. Start the FastAPI backend

uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

API runs at http://localhost:8000. Docs at http://localhost:8000/docs.

3. Start the Streamlit frontend

streamlit run streamlit_app.py

UI runs at http://localhost:8501. Enter a ticker and click Submit.

4. (Optional) Comprehensive Streamlit App

streamlit run streamlit_comprehensive.py

Full dashboard: financials (P&L, Balance Sheet, Cash Flow tables), technicals, sentiment pie chart, AI analyst (Ollama DeepSeek).

5. Run tests

pytest tests/ -v

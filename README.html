📊 Stock Analysis App

Architecture overview, comprehensive dashboard, and run guide

Overview

Scalable stock analysis application with FastAPI backend and Streamlit frontends. Includes a Comprehensive Stock Analysis dashboard with financials (ROCE, ROE), technicals (CAGR, Sharpe, Monte Carlo, ARIMA, HMM), sentiment analysis, and AI perspective via Ollama/DeepSeek.

Architecture

Layered design with dependency injection and repository pattern:

Architecture Flow (Comprehensive App)

User Input (Ticker, Company Name) │ ├─► FinancialRatioService ─► YahooFinancialRepository (balance sheet, P&L, cashflow) │ └─► ROCE, ROE, ROA, margins, Debt/Equity │ ├─► TechnicalAnalysisService ─► StockRepository (Yahoo Finance price data) │ └─► CAGR, Sharpe, Monte Carlo, ARIMA, Holt-Winters, HMM, ML prediction │ ├─► SentimentService ─► GoogleNewsRepository (RSS headlines) │ └─► VADER sentiment → Pie chart (Positive/Neutral/Negative) │ └─► OllamaLLMService (optional) └─► Aggregates financials, news, annual report → BUY/HOLD/SELL analysis via DeepSeek

How to Run

1. Install dependencies

cd project_root
pip install -r requirements.txt

2. Run Comprehensive Streamlit App

streamlit run streamlit_comprehensive.py

UI runs at http://localhost:8501. Enter ticker (e.g. AAPL, MSFT, ALKYLAMINE.NS) and company name for news. Annual report (10-K) fetched from SEC for US tickers.

3. (Optional) Ollama for AI Perspective

# Install Ollama from https://ollama.ai
ollama pull deepseek

Check "Get AI Perspective" in the sidebar to enable.

4. (Optional) Basic App with FastAPI backend

uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
streamlit run streamlit_app.py

5. Run tests

pytest tests/ -v
